# Hospital Data Analysis Using Python

## Project Overview

This project focuses on analyzing hospital patient data using Python to uncover meaningful insights related to patient demographics, disease patterns, hospital admissions, treatment costs, and doctor workloads.

The dataset contains 10,000+ patient records, including patient details, admission information, diseases, doctors, and treatment costs. The analysis helps improve hospital decision-making, resource allocation, and operational efficiency.

## Objectives

* Clean and preprocess hospital data.
* Analyze patient demographics and disease patterns.
* Study admission and discharge trends.
* Evaluate treatment cost distribution.
* Assess doctor-wise patient distribution.
* Generate actionable insights through data visualization.

## Key Findings

* Infection and Fever were among the most common diseases.
* Most admissions were from middle-aged patients.
* Average hospital stay ranged between 3–5 days.
* Cardiac and Critical Care treatments showed higher treatment costs.
* Admissions increased significantly between July and October.
* Doctor workload distribution helped identify resource allocation opportunities.
